# Accounting Reconciliation (Minimal)

A lightweight Next.js web app that lets accounting upload a CSV or Excel file
of Purchase Orders or Sales Orders, fetches the same orders from OrderTime via API,
and validates whether something has been received or shipped.

## Quick Start

### 1. Clone the repo
```bash
git clone https://github.com/<your-username>/acct-recon-min.git
cd acct-recon-min
npm install
